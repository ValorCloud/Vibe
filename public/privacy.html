Privacy Policy

Last updated: June 2026

This app (Vibe Lyricist) is provided as-is and uses only the minimum information needed to operate. We do not sell, share, or permanently store personal data.

Google User Data

Vibe Lyricist offers optional integration with Google Drive so you can open, edit, and save your lyrics files directly from your Drive. This section explains exactly what Google user data is accessed, how it is used, and how it is protected, in compliance with the Google API Services User Data Policy.

OAuth Scopes Requested

When you choose to connect Google Drive, Vibe Lyricist requests one or both of the following OAuth scopes, depending on the operation:

  • https://www.googleapis.com/auth/drive.readonly – Grants read-only access to files in your Google Drive. This scope is used solely to list and download lyrics or audio files you have selected.
  • https://www.googleapis.com/auth/drive.file – Grants read and write access limited strictly to files created or opened by Vibe Lyricist. This scope is used to save or update a lyrics file back to your Drive.

These scopes do not grant access to arbitrary files, folders, or any other data in your Google Account. Access is limited to the files you explicitly pick within the app.

Data Accessed

Through the Google Drive API (v3), Vibe Lyricist may read the following data from your Google Drive:

  • File metadata – file name, file ID, MIME type, and file size of lyrics and audio files (extensions: .txt, .md, .json, .docx, .odt, .mp3, .flac, .wav, .ogg, .m4a, .aac, .opus, .webm, .weba).
  • File content – the text content of lyrics files you open, and the audio stream of audio files you play back via a temporary in-browser blob URL.

No other Google account data (such as your email address, profile information, contacts, calendar, or Drive storage quota) is accessed or requested.

Data Usage

The Google user data accessed by Vibe Lyricist is used exclusively for the following purposes:

  • Opening lyrics files – File content is loaded into the in-app editor so you can read and edit your song lyrics. The content is never transmitted to any server operated by Vibe Lyricist.
  • Saving lyrics files – When you save, the text content of your lyrics is uploaded back to the same file in your Google Drive (or a new file is created). This upload goes directly from your browser to Google's servers via the Google Drive REST API; the data does not pass through our servers.
  • Audio playback – Audio files are streamed directly from Google Drive to your browser and played locally. A temporary blob:// URL is created for the audio element and revoked once playback ends or the file is removed from the player queue.

Google user data is never used for advertising, analytics, AI/ML model training, or any purpose beyond the in-app functionality described above.

Data Storage and Retention

  • OAuth access tokens are stored in browser memory only (JavaScript variables). They are never written to localStorage, sessionStorage, cookies, or any server-side database. Tokens are automatically cleared when you sign out of Google Drive within the app or when you close/refresh the page.
  • Refresh tokens are not requested or stored.
  • File content exists in browser memory only for the duration of your editing session and is discarded when you navigate away or close the app.
  • No Google user data is stored on Vibe Lyricist servers – the application is a client-side single-page app with no backend that handles user data.

Data Sharing

Vibe Lyricist does not share Google user data with any third parties. The only network requests made on your behalf are directly to Google's own APIs (www.googleapis.com, accounts.google.com) using your access token. No Google user data is forwarded to other services or analytics platforms.

Revoking Access

You can revoke Vibe Lyricist's access to your Google Drive at any time from your Google Account permissions page. Once revoked, the app will no longer be able to access your Google Drive until you re-authorize it.

Other Data

Outside of the Google Drive integration, Vibe Lyricist does not collect, transmit, or store personal data. The app runs entirely in your browser. Any lyrics, musical settings, or preferences you create are stored locally in your browser's localStorage and are never sent to our servers.
